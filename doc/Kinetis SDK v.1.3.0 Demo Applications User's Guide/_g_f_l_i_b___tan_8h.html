Data Structure Documentation

struct GFLIB_TAN_TAYLOR_COEF_T_F32

Output of tan( * f32In) for interval [0, /4) of the input angles is divided into eight sub-sectors. Polynomial approximation is done using a 4th order polynomial, for each sub-sector respectively. Four coefficients for a single sub-interval are stored in this (GFLIB_TAN_TAYLOR_COEF_T_F32) structure.
